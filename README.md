# Real360

The Real360 system was built for a Real Estate agency that manages a lot of properties in the US and Mexico.


## Dependencies

- MongoDB
- NodeJS 16
- Axios
- Mongoose
- Typescript 5
- Jest

## Installation

```
 yarn
```
## SEEDING DB

```
 yarn seed
```

## RUN APP

```
yarn start
```

## RUN TESTS

```
yarn test
```
